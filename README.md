## Requirements Libraries
 * scipy                     1.1.0
 * numpy                     1.14.3
 * keras                     2.2.2
 * keras-applications        1.0.4                    
 * imutils                   0.5.1 
 * opencv-python             3.4.3.18
 
## How to train
 * Open Training.ipyb 
 * Change "DATASET_PATH" to path where dataset is kept
 * Run other cells to complete training
## Testing
  * Run run.py with python 3.5
